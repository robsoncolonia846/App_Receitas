# App Receitas (PWA)

App web para salvar receitas da familia, com busca rapida e cadastro simples.

## O que faz
- Salva receitas no navegador (localStorage).
- Cadastro rapido na tela inicial.
- Cadastro completo e edicao na tela `products.html`.
- Historico para marcar receitas feitas.
- Favoritos para acesso rapido.
- Importar e Exportar backup em JSON.

## Estrutura
- `index.html`: tela inicial com busca e cadastro rapido.
- `products.html`: cadastro completo e lista para editar.
- `history.html`: receitas preparadas e pendentes.
- `db.js`: regras de armazenamento e import/export.
- `app.js`: comportamento da interface.
- `styles.css`: estilo visual.

## Publicar no GitHub Pages
1. Suba os arquivos no repositorio.
2. Em `Settings > Pages`, use `main / (root)`.
